# Network Scanning Automation

## Description
Python tools for network reconnaissance using Ping, ARP, and Nmap.

## Files
- ping_scanner.py - Ping scanner for single/multiple hosts
- arp_scanner.py - ARP table viewer with IP-MAC mappings
- nmap_scanner.py - Nmap wrapper with multiple scan types

## Requirements
- Python 3.6+
- Nmap installed (sudo apt-get install nmap)
- net-tools installed (sudo apt-get install net-tools)

## How to Run
python3 ping_scanner.py
python3 arp_scanner.py
python3 nmap_scanner.py

## Note
Only scan networks you own or have permission to scan.
